# 📖 About the Project

PharmaAid Gaza is a mobile healthcare platform developed to improve access to medicines across the Gaza Strip.

The application helps patients search for medicines, check availability in nearby pharmacies, communicate directly with pharmacies using an integrated chat system, place medicine orders, and locate pharmacies through Google Maps.

The project was developed as a Graduation Project using Flutter, Firebase, and Google Maps while following the Waterfall Software Development Model.
