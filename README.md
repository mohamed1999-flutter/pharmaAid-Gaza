# 📖 Overview

PharmaAid Gaza is a modern Flutter-based mobile application designed to improve access to medicines in the Gaza Strip.

The application allows users to search for medicines, check their availability, communicate directly with pharmacies through an integrated chat system, place medicine orders, and locate nearby pharmacies using Google Maps.

The system was developed using Flutter and Firebase while following modern software engineering principles and the Waterfall Software Development Model.
