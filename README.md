# PharmaAid Gaza

## Smart Medicine Availability & Pharmacy Management Platform

### Badges

-   Flutter
-   Dart
-   Firebase
-   Cloud Firestore
-   Google Maps

## About

PharmaAid Gaza is a Flutter-based healthcare application that helps
users search for medicines, communicate with pharmacies through an
integrated chat system, place medicine orders, and locate nearby
pharmacies using Google Maps.
